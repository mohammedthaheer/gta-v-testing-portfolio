Game QA Testing Portfolio – GTA V

This repository showcases my hands-on work in game testing and quality assurance using Grand Theft Auto V (Enhanced version).

The focus of this project is to simulate real-world QA practices by identifying, reproducing, and documenting bugs under controlled test conditions.

🎯 Objectives:
- Identify gameplay, UI, and system-level bugs
- Reproduce issues consistently with clear steps
- Analyze root causes and system behavior
- Document findings in a structured QA format


🧪 Testing is performed using targeted scenarios designed to stress different game systems, including:

- Weapon mechanics and state transitions
- AI behavior under dynamic conditions
- Physics interactions and collision handling
- UI responsiveness under rapid input


🐞 Bug Reports Included
- Weapon Reload Bypass (Multi-Weapon Exploit)


🛠 Skills Demonstrated
- Bug identification and classification
- Reproducibility testing
- Analytical problem-solving
- Clear technical documentation


🎥 Evidence

Each bug report includes:

- Step-by-step reproduction
- Expected vs actual behavior
- Video proof for validation
